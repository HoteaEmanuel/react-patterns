# ExperienceShare

A mini modern type of  social media platform designed for users to share experiences and attend them, engage in discussions and connect with others.

---

## Features

* **Experience Sharing & Discussions:** Post personal experiences and engage in threaded discussions.
* **Follow System:** Connect with other users and customize your home feed based on who you follow.
* **Notification System:** Real-time or activity-based notifications for new followers, replies, and engagement.
* **Type-Safe Routing:** Seamless, robust client-side navigation.

## Tech Stack

* **Frontend:** React, TypeScript
* **Routing:** TanStack Router (Type-safe routing)
* **Database & ORM:** Drizzle ORM
* **TRPC**
* **Styling:** Tailwind CSS
